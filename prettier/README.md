# Prettier Formatter

This directory contains the shared Prettier config used by JS/TS/React/Vue/HTML/Markdown/YAML/GraphQL.

## Install

```bash
npm install -D prettier prettier-plugin-tailwindcss
```

## Use the config

Copy these files into your project root:

- `prettier/.prettierrc` -> `.prettierrc`
- `prettier/.prettierignore` -> `.prettierignore`

## Run

```bash
npx prettier . --check
npx prettier . --write
```
